####ColorUtility
***
Make handling ColorCode more easily.

###Compiling
***
```Bash
git clone https://github.com/Axeryok/ColorUtility.git
cd ColorUtility
gradle build
```
Jars will be located in "build/libs".

###
###Installing
***
